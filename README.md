# ele_shot
2019年全国大学生电子设计大赛,电磁炮.硬件端\stm32

### 说明
同ele_shot 硬件端一样，包含了 电磁炮所有题目实现的控制代码。最终测试的时候 我们依照题目要求做了 一些更改（包括控制方式和数据的更改），但是改动不大。
